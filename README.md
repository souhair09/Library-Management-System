# Library Management System

## Description
A Java library management application for managing books, members, and library operations.

## Features
- Add, update, and delete records
- Manage books and library data
- Search and display information
- Simple user interface
- Basic CRUD operations

## Technologies Used
- Java
- Git
- GitHub

## Project Structure
- `src/` : contains the source code of the application

## What I Learned
- Object-Oriented Programming concepts
- CRUD operations
- Project organization
- Using Git and GitHub to manage code
