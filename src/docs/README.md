# نظام إدارة مكتبة رقمية

الاسم: سهير  
 اللقب: يمبعي  
الفوج: 1

---

## وصف المشروع

نظام لإدارة المكتبات الرقمية يتضمن:
- إدارة عمليات الإعارة والإعادة.
- نظام للبحث والاستعلام عن الكتب.
- سجل كامل لعمليات الإعارة والمرتجع.
- إدارة بيانات الكتب والمستخدمين.

## أهداف المشروع

- تسهيل إدارة المكتبات من خلال نظام إلكتروني.
- تقليل الوقت والجهد في البحث عن الكتب وإدارتها.
- تحسين تجربة المستخدمين في استعارة وإرجاع الكتب.

---

## كيفية تشغيل المشروع

1. تأكد من توفر java jdk اصدار 8 او احدث مثبتا على جهازك
2. قم بتحميل المشروع باستخدام Git:
       git clone : [https://github.com/souhair09/Library-Management-System.git]
3. افتح المشروع باستخدام بيئة تطوير تدعم Java مثل (Eclipse أو IntelliJ أو VS Code).
4. قم بتشغيل الكلاسات الأساسية مثل Main.java لبدء التطبيق.

---

## ملاحظات حول تصميم الكود وتنفيذه

- تم استخدام مبادئ البرمجة الكائنية (OOP) لتقسيم الكود إلى كائنات مثل: كتاب، مستخدم، مكتبة.
- الكود منظم ضمن حزم (Packages) لسهولة التنظيم والصيانة.
- تم استخدام مفاهيم مثل الوراثة (Inheritance) والواجهات (Interfaces) لدعم التوسعة في المستقبل.

---

## معلومات إضافية

- يمكن تعديل الكود لدعم قواعد بيانات خارجية بدلًا من الاعتماد على التخزين المؤقت.
- تمت كتابة المشروع بطريقة تسهّل إضافة مزايا أخرى مثل الإشعارات عند تأخير الإرجاع أو إدارة العضويات.
